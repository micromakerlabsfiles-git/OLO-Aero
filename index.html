1. Firmware Flashing & Screen Setup

1. Switch to the 🔥 Firmware Flasher tab.

2. Connect your device and click Install to program the ESP32-C3 Supermini.

3. Once complete, select your OLED screen type (SSD1306 or SH110X) and save Wi-Fi details.

2. Flight Telemetry Simulator

1. Open the 🎛 Web Controller tab and connect your device. Unlock using passcode admin.

2. Navigate to the 📡 Radar Simulator tab.

3. Drag the red plane marker or click Start Orbit Sim to stream flight data.

4. Tap the device's sensor to toggle between radar sweep and flight telemetry screens.

3. OpenSky Network API & WiFi Setup

• Bounding box is limited to a permanent 10 km query radius around your coordinates.

• Save your OpenSky Client ID and Client Secret in the settings card to avoid anonymous limit (400 calls/day).

• Check API status in the Web Controller or scroll to "OpenSky Status" in the device menu.

4. Device Touch Gestures Reference

• Single Tap: Wake up from sleep, trigger looking animation, or switch Radar & Telemetry screens.

• Double Tap: Open World Clock screen (auto-closes in 10s).

• Triple Tap: Open OLED Settings Menu.

• Long Press (1s): Select/toggle options in OLED Settings Menu.

• Auto-Sleep: Device goes to sleep (sleepy face animation) after 5 minutes of inactivity.

Firmware Programmer

Flash the latest OLO Aero firmware to your ESP32-C3 Supermini directly from your browser. Click the button below to connect your device and begin.

Setup Requirements:

  • Use Google Chrome or Microsoft Edge.
  • Ensure you are in a secure context (HTTPS) like GitHub Pages.
  • Connect the ESP32-C3 Supermini via a data-capable USB cable.
  • Hold the BOOT button on the board while plugging it in to enter bootloader mode if it is not detected automatically.
